# Unity3d-Running Cat

## 概述
结合Ar的小猫跑酷游戏

## 基本功能
### 基本规则
一共分左中右三条轨道，如果前方出现木桥则小猫需要在正确的轨道上通过，否则判定为失败
### 移动
触碰虚拟按钮来移动小猫，需要注意的是显示的方向和现实呈镜像
