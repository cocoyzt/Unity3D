# Unity3d-ArCat

## 概述
用Vuforia实现小猫的demo，并能通过VirtualButton实现和现实的交互

## 功能
摄像头识别到预设的图片dg.jpg时会出现一只小猫
点击两个按钮时，按钮变色，小猫走向被点击的按钮
